1. 从 AstrBot 扩展页直接打开插件 UI 时，编辑器可在无上传任务的全局模版模式下新增或覆盖模版。
2. WebUI 插件页会向编辑器内容页透传 `task`、`name` 等路由查询参数，确保 QQ 中 Bot 发送的编辑器链接可正常保存。
3. 编辑器模版模块将“新增模版”和“选择已有”拆分为独立模式，避免新增和覆盖已有模版混在同一个下拉框里。
4. 编辑器文字样式模块移除自定义字体文件名输入框和字体预览列表，仅保留字体下拉选择。
5. 编辑器将文字区域的 `X`、`Y`、`宽`、`高` 改为更清晰的“左侧位置”“顶部位置”“区域宽度”“区域高度”，并补充拖动画布文字框的说明。
